AIBuilders Art
==============

แพ็คเกจนี้ใช้สำหรับโชว์รูปศิลปะ ascii ซึ่งได้ผลงานมาจากสมาชิกเพจ
AIBuilders ช่วยกันเขียนขึ้นมา

วิธีติดตั้ง
~~~~~~~~~~~

เปิด CMD / Terminal

.. code:: python

    pip install aibuildersart

วิธีใช้งาน
~~~~~~~~~~

.. code:: python

    from aibuildersart import *

    show = UncleEngineer()
    show.art()
    print("Author: {}".format(show.name))
